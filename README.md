## 주의
활발히 개발 중인 유저스크립트입니다. 그렇기에 미완성된 인터페이스, 기능과 버그가 많습니다.

## 설치하기

### 브라우저 별 확장 기능 설치하기
모든 테스트는 Violentmonkey 에서 진행됐습니다, 추가 오류가 발생한다면 이슈 넣어주세요.
